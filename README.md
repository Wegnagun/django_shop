# django_shop
магазин на django

### Технологии: 
Django 4.1.9, django-taggit 3.0.0, Markdown 3.4.1


### Установка: 
#### Windows
`python -m venv venv`

`venv/Scripts/activate`

`python -m pip install --upgrade pip`

`pip install -r requirements.txt`

`в корне проекта создать файл .env и поместить туда данные:`  
`KEY = ваш джанго ключ`  
`DBG = True (или False)`
